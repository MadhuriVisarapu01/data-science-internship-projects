# Data Science Internship Projects

This repository contains the projects completed as part of my Data Science Internship. The projects focus on data analysis, machine learning, data preprocessing, visualization, and database design using real-world datasets.

## Project 1: Zomato Restaurant Analysis

### Objective

Analyze restaurant data from Zomato to uncover insights related to ratings, cuisines, costs, locations, and customer preferences.

### Key Tasks

* Data cleaning and preprocessing
* Exploratory Data Analysis (EDA)
* Feature engineering
* Classification modeling
* Regression analysis
* Clustering and segmentation
* Data visualization

### Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Scikit-learn
